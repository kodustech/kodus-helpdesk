Kodus is a company that provides AI-powered code review tooling. This helpdesk project (kodus-helpdesk) is a monorepo (front + back) built to manage support tickets for Kodus enterprise clients. It provides both an internal view (Kodus team) and a client-facing view (customer workspace).
